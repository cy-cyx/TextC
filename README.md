#####删除

分为三种情况:无孩子结点、一个孩子结点、两个孩子结点

（1）无孩子结点：将结点的双亲结点指向它的指针置NULL，并删除该结点

（2）一个孩子结点：将结点的双亲指向它的指针指向该结点的孩子结点，并删除该结点

（3）两个孩子结点：从左孩子结点中找出最大的结点顶替该结点的位置，并删除该结点

注意:一个要删除无用的结点，不然它会一直存在栈中，内存泄露
